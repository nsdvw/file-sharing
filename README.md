FileSharing
===========
File sharing service written on slimframework.

Installation
------------
1. Clone repository
   ``` sh
$ git clone https://github.com/nsdvw/file-sharing.git
```

2. Composer install
    ``` sh
$ composer install
```

3. Import database
    ``` sh
$ mysql -uuser -ppassword file_sharing < db.sql
```

4. To change mysql dbname, login or password edit config.php file
